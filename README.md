# CRM System

A production-ready full-stack CRM application built with React 19, Vite, Tailwind CSS, Node.js, Express.js, and MongoDB. The project follows a strict 3-layer architecture that separates the Public Website, Client Portal, and Admin Dashboard, ensuring scalability, maintainability, and clean code organization.

## Tech Stack

**Frontend**
- React 19
- Vite
- Tailwind CSS
- React Router
- Axios

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Features

- Public Website
- Client Portal
- Admin Dashboard
- Secure Authentication & Authorization
- Role-Based Access Control
- Responsive UI
- RESTful APIs
- CRM Management
- Analytics & Reporting

## Installation

```bash
# Frontend
cd client
npm install
npm run dev

# Backend
cd server
npm install
npm run dev
```
